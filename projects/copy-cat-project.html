DeltaMath

This is my copy-cat project where I copy another website using my knowledge of html and css. I chose Deltamath as my website because it's a website that looks simple, but tests my understanding of css to make the content look the same.

I came across a few challenges while working on this project. One of these challenges was getting the deltamath logo that I can't find online. I was able to overcome this challenges by learning to snip the picture and moving it into my ide. There were other challenges like positioning my contents. This one was really difficult because the position changes according on the size of the screen. I still wasn't able to completely fix that mistake, so my next step will be using what I learned in responsive web-design to fix that.

Overall, I was pretty satisfied with my result. I was able to practice html and css to build a good looking homepage of a website I like from scratch.
